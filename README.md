# checkARB-Android

[中文版](README_zh.md)

## Overview
This is an Android port of [checkARB](https://github.com/Dere3046/checkARB), utilizing [arb_inspector_jni](https://github.com/Dere3046/arb_inspector_jni) for underlying scanning. It enables inspection of ARB values on Android devices, supporting both non-root and root modes.
